=========================================================================================

You not suppose to modify this directory at all. Any files in it are considered
do-not-touch. The whole contents of this directory is managed automatically by the
"loader-modules.sh" script.

1. You CAN always do is delete everything from this directory if you experience problems
   which ext-manager cannot solve (but keep in mind you will have to reinstall/readd all
   extensions)
2. You SHOULD NOT modify/replace/add files in this directory by hand
3. Please, read the documentation (especially "additional extensions" section) before asking
   questions
4. If you have a problem with a particular extensions you can always run
   "./ext-manager.sh info" to see information about all extensions and where to get more
   info about them/help
